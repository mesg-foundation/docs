The builders' open economy

Get Started →

Open development economy

Exchange services, processes and validations with the community to support every stage of your product lifecycle, no middlemen required.

Multi-language applications

MESG mixes a gRPC API and a container based module approach to offer the most flexibility to developers.

Don’t repeat yourself

Any features, whether centralized or decentralized, can be reused within any application. No heavy reconfigurations required.

Stay in control

Choose exactly how and where your services run, whether on trustless networks, trusted cloud providers, or private servers in any combination.

Scalable applications

The mix of an Event-driven architecture and a decentralized network offers an high scalability and redundancy infrastructure

Decentralized processes

The upcoming decentralized network will enable applications and services to run and execute from entirely within a decentralized cloud.
